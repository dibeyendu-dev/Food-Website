# 🤖 AI Startup Landing Page

A modern and responsive AI Startup Landing Page built using **HTML5, CSS3, and JavaScript**. This project showcases an AI product with a clean design, responsive layout, dark/light mode, smooth scrolling, and interactive JavaScript functionality.

## ✨ Features

- 🌙 Dark / Light Mode
- 📱 Responsive Layout
- 📌 Sticky Navigation Bar
- 📜 Smooth Scroll Navigation
- 📱 Mobile Navigation Menu
- ✨ Navbar Shadow on Scroll
- 🎯 Active Navigation Highlight
- ⌨️ Typing Animation
- 🎴 Feature Card Reveal Animation
- 💰 Pricing Section
- 💬 Auto Testimonial Slider
- 📢 Call To Action Section
- ⬆️ Back To Top Button
- ⌨️ Keyboard Shortcut (Press **T** to Toggle Theme)
- 🎨 Hover Effects on Cards and Buttons
